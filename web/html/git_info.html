🚂 Git-Support für Rocrail

Mit Git kannst du deine plan.xml (Rocrail-Anlagenplan) versionieren und sichern.

🔎 Was ist Git?

Git ist ein Versionskontrollsystem. Es merkt sich jede Änderung an Dateien und ermöglicht es dir, jederzeit zu früheren Versionen zurückzukehren.

💡 Warum für Rocrail?

  • Alle Änderungen am Stellwerk plan.xml werden dokumentiert
  • Du kannst nachvollziehen, wann welche Weiche, Lok oder Block geändert wurde
  • Automatisches Backup bei GitHub (kostenlos, privat oder öffentlich)
  • Teilen deiner Pläne mit anderen – wenn du das möchtest

👤 Für wen geeignet?

Git-Support ist optional. Wenn du deine Anlage einfach nur lokal betreiben möchtest, brauchst du es nicht. Wenn du deine Pläne sichern, versionieren oder mit anderen teilen willst → Git ist perfekt.

⚙️ Wie funktioniert’s?

  1. Klicke im Webinterface auf „Git-Support installieren“
  2. Das System installiert alle benötigten Komponenten
  3. Du bekommst eine Schritt-für-Schritt-Anleitung, wie du den SSH-Key bei GitHub hinterlegst
  4. Danach kannst du im Webinterface:
    • Commits erstellen (Änderungen an plan.xml speichern)
    • Push zu GitHub ausführen
    • Changelog anzeigen

📌 Hinweis

Nur die Datei plan.xml wird versioniert. Alle anderen Rocrail-Dateien (Logs, Backups, Bilder etc.) werden ignoriert.
